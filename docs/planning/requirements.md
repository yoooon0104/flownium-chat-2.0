# MVP2~MVP3 요구사항 정의서 (무상 기능 중심)

업데이트: 2026-03-05

## 0. MVP1 기준선 (현 상태)

MVP2/3 확장은 아래 MVP1 완료 상태를 기준으로 진행한다.

1. 카카오 OAuth + JWT 인증
2. 그룹방 생성/목록/입장
3. 실시간 메시지 송수신 + 히스토리 조회
4. 참여자 online/offline 표시
5. 방 검색 + FAB 생성 모달
6. 로그인/온보딩 분기
7. 우측 상단 플로팅 사용자 메뉴

## 1. 문서 목적

본 문서는 외부 유료 서비스 의존 없이 구현 가능한 기능을 기준으로 단계별 범위(MVP2-A, MVP2-B, MVP3)를 정의한다.

핵심 원칙:
1. 유료 API/SaaS 의존 기능은 제외 또는 보류
2. 필수 보완 항목(P0)은 우선 반영
3. 단계 완료 기준(DoD) 충족 후 다음 단계 진행

## 2. 비용 정책

1. 포함: 무료/자체 구현 가능한 기능
2. 제외: 유료 API/유료 SaaS 의존 기능
3. 보류: 무료 플랜 가능 기능도 즉시 도입하지 않고 추후 검토

## 3. 단계별 범위

## 3.1 MVP2-A (운영/기반)

### 목표

- 운영 배포 가능 상태 확립
- 에러/로그/검증 규칙 표준화 시작

### In Scope

1. Vercel + Render + MongoDB Atlas 배포 안정화
2. 프론트 `VITE_API_BASE_URL` 전환
3. 카카오 운영/로컬 URI 검증
4. 에러 응답 표준화 시작
- `{ error: { code, message, details? } }`
5. 운영 로그 기준 수립
- `info/warn/error` 분류
- 장애 판정 기준 문서화
6. 브랜딩 자산/토큰 1차 적용
- 로고 리소스 경로 및 사용 규칙 정리
- 컬러 토큰(`assets/branding/usage-guide.md`) UI 연동 기준 확정

### Out of Scope

- 친구 도메인 본 구현
- `leave_room` 본 구현
- 인앱 알림 본 구현

## 3.2 MVP2-B (핵심 도메인)

### 목표

- 친구 중심 채팅 모델 전환

### In Scope

1. 친구 모델 구현
- 이메일/닉네임 검색
- 친구 요청/수락/거절/차단
2. 친구 기반 방 생성 강제
- 1:1 direct, 그룹 생성 시 친구 선택 필수
3. `leave_room` 이벤트 + REST 계약
4. 탭 UI 전환
- 친구 목록 / 방 목록
5. 로그인 상태 확인 API
- `/auth/me` 또는 동등 계약

### Out of Scope

- 다국어 본 적용
- 다크모드 본 적용

## 3.3 MVP3 (무상 특화 기능 확장)

### 목표

- 사용자 체감 기능 고도화

### In Scope

1. 인앱 알림
- 멘션/새 메시지/초대/읽음 상태 배지
2. 메시지 검색 고도화
- 사람/기간/키워드 필터
3. 북마크/핀
4. 읽음 상태 정교화
5. 다국어(i18n)
6. 다크모드
7. 보안 보완
- 기기 세션 목록, 로그인 이력

### Out of Scope

- 유료 AI 요약/액션 API
- 유료 번역 API
- 유료 푸시/문자/메일 서비스
- 유료 분석/모니터링 SaaS

## 4. 필수 보완 항목 (P0)

1. 에러 코드 표준화 (`error.code`)
2. 친구/권한 상태 전이 모델 명세
3. 알림 도메인 모델 분리
4. 운영 로그/장애 기준 문서화

## 5. MVP2-B 상세 기능 요구

1. FR-B01: 친구 미관계 direct 생성 차단
2. FR-B02: leave_room 후 참여자/권한 상태 일관성
3. FR-B03: 친구/방 탭 전환 UX 정상
4. FR-B04: 표준 에러코드 반환
5. FR-B05: 친구 검색은 이메일/닉네임 키워드 지원

## 6. 수락 기준 (DoD)

## MVP2-A
1. 운영 도메인에서 카카오 로그인 성공
2. 방 생성/입장/메시지/참여자 흐름 정상
3. CORS/소켓 재연결 루프 문제 없음
4. 배포 체크리스트 수행 기록 문서화

## MVP2-B
1. 친구 미관계 direct 생성 차단
2. leave_room 후 참여자/권한 일관성 유지
3. 친구/방 탭 전환 UX 정상
4. 표준 에러코드 반환

## MVP3
1. 인앱 알림 생성/읽음 동기화 정상
2. 검색/북마크/읽음 기능 회귀 없음
3. 다국어/다크모드 전환 시 UI 깨짐 없음
